# examples/

本目录存放示例项目结构和使用说明，帮助理解如何在实际项目中应用本模板仓库。

## 目录结构

```text
examples/
└── simple-project/     # 最简单的单语言项目示例
```

## 目的

- 展示下游项目同步模板后的推荐目录结构
- 展示 `AGENTS.md`、`skills/`、`templates/` 等文件如何在实际项目中配合使用
- 作为新接入项目的参考起点

## 使用方式

1. 参考对应示例中的目录结构和文件说明
2. 运行 `tools/agent-template/sync-template.sh` 初始化你的项目
3. 根据项目实际情况定制 `AGENTS.md` 等本地文件
